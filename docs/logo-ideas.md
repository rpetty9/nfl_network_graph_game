# Logo Ideas

Paused logo direction on March 17, 2026.

Most promising direction so far:

- a simple five-node graph mark based on the actual game board shape
- five outer nodes only, no extra center node
- all ten node-to-node connections visible
- blue-to-violet gradient lines and nodes
- small center medallion inspired by the in-game relationship circle, with `5W`

Notes from the last pass:

- shield/playbook variants were dropped in favor of the simplest graph-based mark
- smaller nodes looked better than larger gold ones
- horizontal links were tricky to read at small sizes, so any future pass should test visibility at favicon/app-icon scale early
- keeping the mark text-free, or nearly text-free, felt cleaner than a full `Five Wide` wordmark inside the icon

If we revisit this later, good next directions would be:

- a polished single-mark SVG based on the current graph layout
- a proper app-icon tile version and a separate horizontal wordmark version
- a version that borrows the styling language from the in-game center circle
